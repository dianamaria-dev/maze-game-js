# Pixel Heart Maze
A cute interactive maze game built with HTML, CSS, and JavaScript.

**Play here:**
https://dianamaria-dev.github.io/maze-game-js/ 

## Features:
- Mobile + keyboard controls
- Key collection system
- Animated heart effects
- Responsive design (works on phone + laptop)

## Tech used:
- HTML
- CSS
- JavaScript
